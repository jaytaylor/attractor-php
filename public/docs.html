Attractor PHP Dashboard Docs

This page is fully served by the application with no external dependencies.

Views

  • Monitor: run list/details, logs, graph, artifacts, human-gate actions
  • Create: DOT edit/validate/preview/run, DOT generate/fix/iterate
  • Archived: archived runs with unarchive/open
  • Docs: this documentation page

Core API

  • GET /api/v1/pipelines
  • POST /api/v1/pipelines
  • GET /api/v1/pipelines/{id}
  • GET /api/v1/pipelines/{id}/events (SSE)
  • GET /api/v1/events (SSE)
  • POST /api/v1/dot/generate/stream (SSE)
  • POST /api/v1/dot/fix/stream (SSE)
  • POST /api/v1/dot/iterate/stream (SSE)

Error Envelope

{ "error": "human readable message", "code": "MACHINE_CODE" }
