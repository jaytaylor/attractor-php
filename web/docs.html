Core API

  • GET /api/v1/pipelines, POST /api/v1/pipelines
  • GET /api/v1/pipelines/{id}, POST /api/v1/pipelines/{id}/cancel
  • GET /api/v1/pipelines/{id}/events, GET /api/v1/events
  • GET /api/v1/pipelines/{id}/questions, POST /api/v1/pipelines/{id}/questions/{qid}/answer
  • POST /api/v1/dot/generate/stream, POST /api/v1/dot/fix/stream, POST /api/v1/dot/iterate/stream

SSE Contract

Each SSE response is text/event-stream and sends newline-delimited data: {json} frames.

For incremental polling, both event endpoints accept optional ?sinceTs=<epoch_ms> and return only newer events.

data: {"type":"Snapshot","payload":{...}}

data: {"runId":"run-...","type":"StageCompleted","payload":{...}}
